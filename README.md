# Projeto HemoCamp

HemoCamp é um projeto integrador, iniciado pelos estudantes da PUCC (Pontifícia Universidade Católica de Campinas).
  Nasceu a partir de uma carência existente no Brasil em geral: doação de sangue. Vimos que existem muitas notícias
    acerca da falta de doação, hospitais com estoques críticos.
  Então, vimos com uma oportunidade de explorar um campo em falta, imaginamos um site no qual hemocentros possam
    se centralizar para tornar a divulgação prática, sem necessidade de verificar se x hemocentro precisa ou se 
    y está com carência. Assim, queremos tornar a informação de fácil acesso.
  

As principais linguagens utilizadas foram:
  - HTML
  - CSS
  - JavaScript
  - Python
  - MySQL

Importações essenciais (obs.: é necessário ser o adm de sua máquina para que algumas bibliotecas funcionem devidamente):
  - flask
  - FLASK_CORS
  - dotenv
  - datetime
  - secrets
  - re
  - jwt
  - bcrypt
  - functools
  - os
  - sib_api_v3_sdk
  - sib_api_v3_sdk.rest
  - pathlib
  - mysql.connector
  - mysql
 *Consegue baixar no terminal de sua máquina, sendo windows, através deste comando: pip install flask flask-cors python-dotenv PyJWT bcrypt sib-api-v3-sdk mysql-connector-python
